# Security Policy

## Supported Versions

| Version   | Supported          |
| --------- | ------------------ |
| >= 1.12.x | :white_check_mark: |
| < 1.12.0  | :x:                |

## Reporting a Vulnerability

Please contact security@getformwork.org with an explaination of the security issue you found and we'll work together to resolve it.
